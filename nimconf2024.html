Karax, Kraut, Karkas

Create compact, fast, and composable frontend and have fun doing it

Key Points

  1. Karax is a great framework: fast, lightweight, and stable but it lacks some sugar, namely routing and components.

  2. Kraut is the missing routing layer for Karax.

    • Sauer is a helper CLI tool that makes adding routes trivial.

  3. Karkas is a library of layouting primitives.
